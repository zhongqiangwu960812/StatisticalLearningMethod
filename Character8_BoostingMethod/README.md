## 支持向量机
> 本节介绍：
>> * 本节介绍了提升方法， 提升（boosting）方法是一种常用的统计学习方法，在分类问题中，它通过改变训练样本的权重，
学习多个分类器，并将这些分类器进行线性组合，提高分类性能
>> * 提升方法的思路：将弱可学习算法提升为强可学习算法
（即从一个弱学习算法出发，通过反复学习不同的参数，得到一系列基本分类器，通过组和
这些弱分类器，构成一个强分类器）
>
